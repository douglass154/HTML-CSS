Príncipais formatações:

Negrito / Destaque

Nessa frase temo um termo em negrito usando a tag <b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> (semântica)

A tag " <b> " não é para se utilizar, pois ela não é semântica.
Já a tag "<strong>" é a certa a se utilizar, pois ela sim é semântica.

Itálico / Ênfase

Nesta frase, temos um "termo em itálico" usando a tag <i> (Não semântica)

Nesta frase temos um "termo em ênfase" usando a tag <em> (semântica)

A tag " <i> " não é para se utilizar, pois ela não é semântica.
Já a tag "<em>" é a certa a se utilizar, pois ela sim é semântica.

Texto marcado

podemos criar também um texto marcado usando a tag "<mark>"
E mudar a cor da marcação colocando um "style" dentro da tag ou no "head" do código.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag "<u>" (Não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x20+3

Texto subscrito

Para inserir coisas do tipo: H2O
