
                                                          ==  CÍRCULO CROMÁTICO  ==


  Cores Primárias = 
==> Azul, Amarelo e Vermelho


  Da junção das cores primárias, temos as três cores secundárias que são = 
==> Laranja (amarelo + vermelho), Violeta/Roxo (Azul + Vermelho) e o Verde (Azul + Amarelo)


  Da junção de uma cor primária com uma secundária, temos as seis cores terciárias =

==> Amarelo-esverdeado (amarelo+verde)
==> Amarelo-alaranjado (amarelo+laranja)
==> Vermelho-alaranjado (vermelho+laranja)
==> Vermelho-arroxeado (vermelho+roxo)
==> Azul-arroxeado (azul+roxo)
==> Azul-esverdeado (azul+verde)

(Note: Cores terciárias são representadas primeiro pelo nome das cores primárias (amarelo) e logo em seguida representada por uma cor secundária (verde), ficando amarelo-esverdeado)


                                                           ==  TEMPERATURA DE CORES  ==


  Cores Frias: estão associadas a sensações mais calmas, de frescor e tranquilidade =
==> Amarelo-esverdeado, Verde, Azul-esverdeado, Azul, Azul-arroxeado e Roxo


  Cores Quentes: As cores quentes, criam uma sensação de calor e proximidade =
==>  Amarelo, Amarelo-alaranjado, Laranja, Vermelho-alaranjado, Vermelho e Vermelho-arroxeado


                                                           ==  CORES COMPLEMENTARES  ==


São aquelas que apresentam o maior contraste entre si. Elas estão localizadas do lado imediatamente oposto do círculo cromático.

Se pegarmos qualquer cor primária, a sua cor complementar é sempre uma cor secundária. De forma similar, qualquer cor terciária tem uma outra cor terciária como complementar. Quando juntamos duas  cores complementares, sempre obtemos o cinza.


                                                                ==  CORES ANÁLOGAS  ==


Diferente das cores complementares, que estão do lado extremo oposto no círculo cromático, as cores análogas são aquelas que são imediatamente vizinhas entre si. 

Por serem cores consecutivas, as cores análogas possuem um baixo contraste entre elas, mas criam uma bela harmonia quando combinadas em um mesmo design.

                                                             ==  PESO DAS FONTES  ==


Para definir o peso das fontes no CSS utilize a propriedade "font-weight"


Os valores a ser definidos no tamanho são:

==> Lighter
==> Normal
==> Bold
==> Bolder
==> 100 -> 200 -> 300 -> ... 900


Uma propriedade para se utilizar e colocar todos os valores em uma única linha, a propriedade é "font"
Coloque os valores nesta ordem, é obrigatório esta ordem:

==> font-style  ->  font-weight  ->  font-size  ->  font-family

==> Exemplo: (==> font: italic bolder 2em Helvetica, sans - serif;)


                                                      ==  FONTES EXTERNAS BAIXADAS  ==


Para linkar a fonte baixada no seu CSS, utillize a regra "@font-face". Isso irá criar um:

font-family:
src: url()

dentro de "src: url()" coloque os nomes identicos da fonte como está no arquivo e com os mesmos espaçamentos e vírgulas que estiverem logo abaixo, exemplo:

font-family:
src: url('') format('')

Como no exemplo abaixo:

@font-face {
    font-family: 'upheav';
    src: url('fonts/upheavtt.ttf') format('truetype'), url('fonts/Upheaval.fon') format('fon');


 Tipos de format()

==> opentype (otf)
==> truetype (ttf)
==> embedded-opentype
==> truetype-aat (Apple Advanced Typography)
==> svg
